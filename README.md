# SIGNAL LOST

> Game narrative tương tác chạy trên trình duyệt, được trình bày qua giao diện điện thoại mô phỏng.

---

## 1. Cài Git

Kiểm tra Git đã có chưa:

```bash
git --version
```

Nếu chưa có, tải và cài tại [git-scm.com](https://git-scm.com), sau đó chạy lại lệnh trên để xác nhận.

---

## 2. Clone dự án

```bash
git clone https://github.com/NMHx2005/Thu_Bau_04_05_2026_9.git
```

Lệnh này tạo thư mục `Thu_Bau_04_05_2026_9` tại vị trí hiện tại. Vào thư mục đó:

```bash
cd Thu_Bau_04_05_2026_9
```

---

## 3. Cài VS Code

Tải và cài VS Code tại [code.visualstudio.com](https://code.visualstudio.com). Chọn bản phù hợp với hệ điều hành (Windows / macOS / Linux) rồi cài như bình thường.

---

## 4. Cài extension Live Server

1. Mở VS Code
2. Vào panel **Extensions** — nhấn `Ctrl + Shift + X` (Windows/Linux) hoặc `Cmd + Shift + X` (macOS)
3. Tìm **Live Server**
4. Chọn extension của tác giả **Ritwick Dey** → nhấn **Install**

---

## 5. Chạy dự án

1. Trong VS Code, vào `File → Open Folder…` → chọn thư mục `signal-9`
2. Trong panel **Explorer** bên trái, click chuột phải vào `index.html`
3. Chọn **Open with Live Server**
4. Trình duyệt tự mở tại `http://127.0.0.1:5500/index.html`

Để dừng server, nhấn vào **Port: 5500** ở góc dưới bên phải VS Code → chọn **Stop Server**.
